Place your APK (e.g. app-release.apk) and version.json in this folder for Firebase Hosting deployment.

- version.json example:
  {
    "version": "1.0.0",
    "apk_url": "https://vip-lounge-f3730.web.app/app-release.apk"
  }
- APK file should be named app-release.apk or match the URL in version.json.
